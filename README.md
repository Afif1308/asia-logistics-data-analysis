# Asia Logistics Performance & Supply Chain Insights
### Analyzing Efficiency and Profitability

## 📌 Project Overview
Analisis mendalam terhadap 31.000+ transaksi logistik di Asia. Proyek ini mendemonstrasikan transisi dari pemrosesan spreadsheet konvensional ke analisis berbasis **Python** untuk menangani data skala besar secara akurat.

## 📂 Repository Structure
Repositori ini disusun secara sistematis untuk memudahkan navigasi:

- 📁 **Dashboard/**: Berisi file visualisasi Tableau yang interaktif (Tableau Workbook).
- 📁 **Data_Cleaning/**: Dataset yang telah diproses dan siap digunakan dalam format `.csv` dan `.xlsx`.
- 📁 **Notebook/**: Dokumentasi teknis pembersihan dan analisis data menggunakan **Google Colab (.ipynb)**.
- 📁 **Raw_Data/**: Dataset asli sebelum dilakukan pemrosesan (Historical raw data).

## 🛠️ Tech Stack & Methodology
- **Data Wrangling:** Python (Pandas & NumPy) via Google Colab.
- **Visual Analytics:** Tableau.
- **Key Metric:** Shipping Gap (Efficiency) & Weighted Average (Profitability).

## 🚀 Key Highlights
1. **Scalability Solution:** Mengatasi kendala performa Excel pada 31.000+ baris data menggunakan skrip Python.
2. **Data Integrity:** Membersihkan anomali tanggal dan nilai penjualan negatif melalui pipeline otomatis.
3. **Business Insight:** Menemukan margin keuntungan riil wilayah Asia berada di kisaran **11%**.
